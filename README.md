# 串口数据读取上位机

基于 PyQt6 的串口原始数据读取程序，将接收到的数据保存为 txt 文件（带时间戳的 hex 格式）。

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 使用说明

1. **模拟串口模式**：勾选「模拟串口模式」可不连接任何硬件，程序从 `simulate_input.txt` 读取模拟数据，按所选波特率（默认 115200）的传输时间间隔发送
2. 选择串口和波特率（模拟模式下可忽略）
3. 点击「打开串口」连接设备
3. 点击「开始记录」开始将数据写入 `historydata/` 文件夹
4. 点击「停止记录」结束写入（会写入结束分割线）
5. 点击「关闭串口」断开连接（若正在记录会先写入结束分割线再关闭）

### 模拟串口数据源 `simulate_input.txt`

- 每行作为一条模拟数据，按顺序循环发送
- 支持 **hex**：`01020304` 或 `01 02 03 04`
- 支持 **文本**：`Hello`（按 UTF-8 编码）
- 以 `#` 开头的行会被忽略

## 输出格式

文件保存在 `historydata/` 目录，文件名格式：`YYYY-MM-DD_HH-mm-ss.txt`（日期+开始时间）。

文件内容用分割线标注开始/结束时间及设备信息：

```
============================================================
开始时间: 2025-03-09 10:00:00  设备: COM3  波特率: 115200
============================================================
[2025-03-09 10:00:00.123] 48656c6c6f
[2025-03-09 10:00:00.456] 576f726c64

============================================================
结束时间: 2025-03-09 10:05:30
============================================================
```
